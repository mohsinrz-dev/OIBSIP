# ⚡ Mohsin Raza — Smart Task Manager

A professional productivity dashboard built with pure HTML, CSS, and JavaScript. Designed to look and feel like a real SaaS product — not a basic college to-do list.

---

## What It Does

This is a fully functional smart task manager that runs entirely in the browser with zero setup. Everything you add is saved automatically and persists after page refresh using localStorage.

**Task Management**
- Add, edit, and delete tasks
- Mark tasks complete and undo them
- Assign a category, priority level, due date, and notes to every task
- Drag and drop tasks to reorder them

**Smart Features**
- Live search across task titles and notes
- Filter by priority (High / Medium / Low)
- Sort by newest, oldest, A–Z, or priority
- Filter by view: All, Today, Upcoming, Completed
- Filter by category from the sidebar

**Dashboard**
- Animated stats strip: total, completed, pending, completion rate
- Overall and today's progress bars that animate on load
- Productivity score calculated from completion rates
- Activity feed logging every action with timestamps
- Live clock and date display
- Daily rotating motivational quote

**UX Details**
- Dark / Light mode toggle (preference saved)
- Floating Action Button to quickly focus the input
- Toast notifications for every action
- Keyboard support: Enter to add, Escape to close modal
- Fully responsive for mobile, tablet, and desktop

---

## How to Run

1. Download this folder
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari)
3. That's it — no install, no server, no build step needed

All data is stored in your browser's localStorage and survives page refreshes.

## Technologies Used

| Layer | Details |
|---|---|
| Structure | HTML5 (semantic sections) |
| Styling | CSS3 — custom properties, glassmorphism, keyframe animations, CSS Grid & Flexbox |
| Logic | Vanilla JavaScript (ES6+) — localStorage, drag-and-drop API |
| Typography | Poppins + Inter via Google Fonts |
| Graphics | HTML5 Canvas (floating particle system) |

No frameworks. No libraries. No build tools.
