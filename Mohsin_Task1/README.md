# ⚛ Mohsin Raza — Scientific Calculator

A futuristic scientific calculator built with pure HTML, CSS, and JavaScript. Designed to look and feel like a premium software product with a Neon Cyber Glassmorphism aesthetic.

---

## What It Does

This is a fully functional scientific calculator that runs entirely in the browser — no installs, no dependencies, no internet required after download.

**Basic operations:** addition, subtraction, multiplication, division, percentage, and decimal support.

**Scientific functions:** sin, cos, tan (in degrees), log₁₀, ln, square root, square (x²), power (xⁿ), factorial (n!), and π.

**Extra features:**
- Calculation history panel — stores your last 20 results and lets you tap any entry to restore it
- Copy button — copies the current result to your clipboard instantly
- Full keyboard support — type numbers and operators directly, use Enter to evaluate, Esc to clear, and Backspace to delete
- Dark / Light mode toggle

---

## How to Run

1. Download this folder
2. Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge)
3. That's it — no build step, no server needed
